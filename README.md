# FlowOps

FlowOps is a smart operations platform built for service-based businesses.

It centralizes booking management, intake tracking, and revenue intelligence into one unified dashboard.

## Key Features

- AI-driven onboarding for business configuration
- Booking lifecycle management (Pending → Confirmed → Completed)
- Client intake tracking with completion rate monitoring
- Revenue analytics and average booking value calculation
- Operational system health indicator
- Lead management
- Inventory tracking

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- API route architecture

## Architecture

FlowOps separates:
- Public booking and contact interfaces
- Operational dashboard
- Modular API routes
- AI onboarding configuration layer

For hackathon scope, data is stored in-memory. In production, this would connect to persistent storage.
